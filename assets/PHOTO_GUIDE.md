## Tips
- All images should be JPG or PNG format
- Keep file sizes under 500KB each for fast loading
- Square photos work best for team headshots
- Landscape (wide) photos work best for hero/banner slides
- If logo.png is missing, the site auto-shows the text logo fallback

## Updating Contact Info
Open each HTML file and search for:
- `nyrcst@nrcs.org` → replace with your actual email
- `+977-1-5370650` → replace with your actual phone number
- Social media links → update `href="#"` with your actual Facebook, Instagram, YouTube URLs
